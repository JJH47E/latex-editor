What is this?

This is a LaTeX editor written in Typescript that makes use of both Angular & Electron frameworks to create a desktop application for editing LaTeX documents.

What is LaTeX?

LaTeX is a document preparation system used for creating documents, such as scientific papers, academic reports, and books. It was developed in the 1980s by Leslie Lamport and is based on the TeX typesetting system created by Donald Knuth.

How do I Get Started?

You can start by taking a look at any of the included templates. Alternatively, you could read a LaTeX guide such as this to learn the basics.
